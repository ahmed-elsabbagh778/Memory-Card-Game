# Memory card Game

## Overview
The Memory card Game is a fun and interactive game where players flip cards to find matching pairs. The game keeps track of the score, time, and number of moves. It is built using *HTML, CSS, and JavaScript*.

## Features
- Flip cards to reveal hidden symbols
- Match pairs to win the game
- Score tracking system
- Timer to track game duration
- Move counter to measure efficiency

## Technologies Used
- HTML – Structure of the game
- CSS – Styling and animations
- JavaScript – Game logic and interactivity

## How to Play
1. Click on any card to reveal its symbol.
2. Click on another card to find a matching pair.
3. If the cards match, they stay flipped. If not, they flip back after a short delay.
4. Continue until all pairs are matched.
5. Track your performance with the score, timer, and move counter.
